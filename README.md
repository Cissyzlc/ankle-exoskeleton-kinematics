# Ankle Exoskeleton Kinematic Analysis

This repository contains MATLAB code used for motion capture processing and kinematic analysis in an ankle exoskeleton project.

Users should replace BASE_DIR with their local data path.

## Features

- Functional axis identification (PF/DF and Inv/Ev)
- Exact Grood–Suntay decomposition
- Linear BCH approximation
- Anchor point evaluation

## Usage

Run the main script:

MoCap_AxesIdentification_ExactAngles.m

## Notes

- Motion capture data is not fully included due to file size constraints.
- Figures in the report can be reproduced using the provided scripts.
